# Pacumuto de Jiba

Pagina comercial independiente para JIBA dentro del repositorio `Prueba2`.

## Como verla

Cuando esta rama se publique o se mezcle a `main`, la pagina quedara disponible como:

`https://tikio2025.github.io/Prueba2/jiba/`

## Que incluye

- Portada comercial para Pacumuto de Jiba.
- Menu con filtros por parrilla, almuerzos y bebidas.
- Botones que agregan platos al formulario de pedido.
- Formulario que abre WhatsApp con mensaje listo.
- Secciones de promos, galeria y contacto.
- Diseno responsive para celular y escritorio.
