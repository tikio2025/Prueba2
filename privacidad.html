Información del sitio

Privacidad, explicada sin rodeos.

Esta página describe qué guarda la vitrina y qué ocurre cuando decides continuar en una plataforma externa.

Resumen: la lista se guarda en tu navegador; este sitio no procesa pagos ni crea cuentas; tú decides si envías el mensaje por WhatsApp.

Lista de consulta

Cuando agregas productos, ajustas cantidades o escribes una nota, esos datos se guardan mediante localStorage en el navegador y el dispositivo que estás usando. Sirve para conservar tu lista entre visitas.

La lista no se envía automáticamente a Vendedor SCZ. Puedes eliminar productos o vaciarla desde “Mi lista”. También puedes borrarla eliminando los datos del sitio desde la configuración de tu navegador.

Pagos y cuentas

Este sitio no procesa pagos, no solicita datos de tarjetas y no ofrece registro de usuarios. La vitrina prepara una consulta; cualquier dato comercial se confirma fuera del sitio.

Contacto por WhatsApp

Al pulsar “Abrir WhatsApp”, el sitio genera un enlace con el texto de tu lista y te lleva a WhatsApp. El mensaje solo se envía si tú completas esa acción en la plataforma.

Los datos que compartas allí también estarán sujetos a las condiciones y políticas de WhatsApp. Conviene revisar el mensaje antes de enviarlo y evitar información que no sea necesaria para la consulta.

Enlaces externos y alojamiento

El sitio puede incluir enlaces a WhatsApp y Facebook. Al abrirlos, pasas a servicios externos con sus propias políticas. La web está publicada mediante GitHub Pages, cuyo servicio puede manejar información técnica necesaria para entregar el contenido.

Cambios en esta información

Si el funcionamiento del sitio cambia, esta página deberá actualizarse para describirlo. Este texto es informativo y no sustituye una revisión legal profesional.
